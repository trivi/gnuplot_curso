6. La plantilla: esa gran amiga

﻿
Jue, 10/07/2008 - 15:59 — grupo

A menudo es necesario hacer gráficas distintas para conjuntos de datos distintos pero con el mismo, o casi el mismo título y opciones (ajustes, etc). Así que es muy útil echar mano de las plantillas. Archivos de configuración estándar que adaptamos a cada situación.
Veámos la siguiente situación: debes hacer tres ajustes lineales de tres conjuntos de datos y presentar las tres gráficas que sólo difieren en el título (Gráfica 1, 2 y 3, respectívamente). Pues echas mano de una sola plantilla y la adaptas a los tres casos:

set encoding iso_8859_1
set xlabel 'Tiempo t (s)'
set ylabel 'Voltaje v_{c}(t)'
set title 'Gr\áfica 1: Voltaje en el condensador frente al tiempo.'
set xrange[0:450]
plot y(x) title 'teórico', 'datos1.txt' u 1:2:3:4 w boxxyerr notitle, 'datos1.txt' u 1:2:3:4 w xyerr title 'experimental'
pause -1
set term postscript eps enhanced
set size 2.1,2.1
set output '1.eps'
replot

Donde los campos en negrita son los que cambiamos de un archivo .plt a otro, es útil tener un .plt para cada archivo de datos por si tienes que volver a hacerlo pasado un tiempo, cuando ya ni te acuerdas de como querías representarlo y debes hacerlo de nuevo.
Aquí se muestra un modelo estándar con algunas líneas comentadas por si hacen falta en algún momento ser utilizadas, puedes descargarla haciendo clic aquí:

set encoding iso_8859_1
set xlabel 'T\ítulo eje X'
set ylabel 'T\ítulo eje Y'
set title 'T\ítulo de la gr\áfica'
# Para quitar las leyendas .
# set nokey
# set xrange [0:10]
# set yrange [0:10]
# set zeroaxis
# Ajustar y(x)=a*x+b
# y(x)=a*x+b
# fit y(x) 'datos.dat' using 1:2 via a,b
plot y(x), 'datos.dat' u 1:2:3:4 w boxxyerr title 'datos', "datos.dat" u 1:2:3:4 w xyerr notitle
pause -1
set term postscript eps enhanced
set size 2.1,2.1
set output 'grafica1.eps'
replot

Donde pause -1 espera a que pulsemos enter para seguir adelante, para que demos el visto bueno a la gráfica y replot, tras elegir el terminal postscript, crea un archivo grafica1.eps con la gráfica.
Cabe destacar que los caracteres griegos y matemáticos que no salen en la ventana de consola saldrán en el fichero .eps ya que es ahí donde activamos el enhanced comentado anteriormente.

Si dieran problemas los acentos, considera no incluirlos.
