4. Configurando el gráfico (settings)

﻿
Mar, 08/07/2008 - 14:24 — grupo

Todas las órdenes que configuran el gráfico (su aspecto) suelen empezar por set o unset y se escriben antes de pintar la gráfica.

  • Puesta a cero de la configuración, desconfiguración.
  • Rangos de representación de los ejes X, Y y Z (este último en el caso de representación 3D).
  • Título de la gráfica y etiquetas en los ejes X, Y y Z.
  • Representación de los ejes cartesianos.
  • Representando los ejes en escala logarítmica.
  • Acentos y carácteres latinos.
  • Leyendas en la gráfica.
  • Terminal (donde se representa).
  • Carácteres matemáticos y griegos.
  • Borrando una determinada configuración.
  • Modificando el color de fondo de la gráfica.
  • Creando pausas entre órdenes.
  • Creando comentarios e ignorando líneas (para archivos con órdenes, plantillas).

  • Puesta a cero de la configuración, desconfiguración:

    reset

    Esta orden elimina cualquier configuración previa.

  • Rangos de representación de los ejes X, Y y Z (este último en el caso de representación 3D):

    set xrange [0:1000]

    Esta orden restringe el rango de representación del eje x entre los valores 0 y 1000. Análogamente para los demás ejes

    set yrange [1:259]
    set zrange [-5:3]

    Para eje Y y Z, respectívamente.

  • Título de la gráfica y etiquetas en los ejes X, Y y Z:

    set title 'Este es el título.'

    Esta orden escribe el texto que está entre comillas como título de la gráfica.

    set xlabel 'Este es el eje x.'

    Esta orden escribe en el eje X el título que está entre comillas, análogamente para los demás ejes

    set ylabel 'Este es el eje x.'
    set zlabel 'Este es el eje x.'

  • Representación de los ejes cartesianos:
    Si se desea que aparezcan los ejes de coordenadas podemos escribir

    set xzeroaxis

    Para el eje X. Análogamente para el eje Y se escribe

    set yzeroaxis

    Existen opciones para personalizar los ejes pintados con esta orden, dejamos esas opciones para consultarlas en la ayuda del programa, ejecutando dentro del programa:

    help zeroaxis

  • Representando los ejes en escala logarítmica:
    Es muy útil esta opción cuando se están realizando los diagramas de bode en el laboratorio de electromagnetismo.

    set logscale x 10

    Esto representa el eje x en escala logarítmica en base 10. Se puede poner también para el eje Y y Z, cambiando

    set logscale y/z 2/10

    donde 2 es la escala logarítmica en base 2 y si no se pone número alguno es el logaritmo neperiano (natural).

  • Acentos y carácteres latinos (no a todo el mundo le funciona):
    Debido a que el programa está escrito en Inglés, para dotar de los carácteres latinos y acentos a los textos (que de otra forma dan errores) ha de ponerse la siguiente orden (la pondremos la primera de todas)

    set encoding iso_8859_1

    donde iso_8859_1 es la fuente más utilizada para los carácteres latinos. Si no es el que conviene o en el sistema tiene otro nombre es posible consultar las distintas "fuentes" disponibles ejecutando dentro del programa

    help set encoding

    Para añadir un carácter con acento, por ejemplo en la o de neutrón, escribimos

    set title 'Vida media del neutr\ón'

  • Leyendas en la gráfica:
    Por defecto, cuando se pinta un gráfica, aparece una leyenda en la esquina superior derecha con un título que podemos configurar en la orden plot, que veremos más adelante.

    set key off

    Con esta orden se deshabilitan las leyendas. También es posible colocarlas en otro sitio, si es que queremos o estorban.

    set key top/bottom right/left

    Según queramos que esté arriba/abajo y a la derecha/izquierda. También podemos colocarla en otros sitios

    set key below

    Para ponerlo debajo del eje x (y debajo del título del eje, si es que hay), ó bien

    set key outside top/bottom

    para ponerlo fuera de la gráfica, a la izquierda y arriba/abajo.

  • Terminal (donde se representa):
    Si no especificamos nada, por defecto el terminal escogido es una ventana de consola. Pero es posible escoger como terminal un archivo .png, .jpg, .eps, etc.
    Para saber de cuales terminales disponemos, los cuales varían de un sistema a otro, ejecutamos dentro del programa

    help set terminal

    Donde, entre otras cosas, aparecerá una lista de las terminales posibles, aunque muchas pueden fallar si no están bien configuradas en el sistema operativo.
    En Linux, por defecto está la x11 (ventana consola). De modo que si queremos que salga una imagen en png escribimos

    set terminal png

    Y a continuación especificamos el tamaño de la "hoja" y el nombre del archivo en que se guardará

    set size 2.1,2.1
    set output 'imagen.png'

    2.1,2.1 es el tamaño A4, por defecto viene en tamaño 1,1. Y la imagen se guardará en un archivo llamado imagen.png dentro de la carpeta en la que se ejecute el programa.

  • Carácteres matemáticos y griegos:
    También es posible añadir la opción enhanced que existe en algunos terminales (como "x11" y "postscript eps") y en otros no (como "png" y "jpeg") y que permite escribir superíndices, subíndices y el alfabeto griego, este último solo on "postscript eps".

    set term x11/postscript eps enhanced

    Entonces en el texto podemos apreciar lo siguiente:

    Código Resultado
    a^{2} a2
    a_{2} a2
    {/Symbol a} α
    {/Symbol a}_{/Symbol b}^{2} αβ2
    {/Symbol a}_{{/Symbol b}^{2}} αβ2

    Usualmente usaremos la terminal postscript.

  • Borrando una determinada configuración:
    Para borrar una determinada configuración se utiliza el comando “unset” antepuesto al nombre del comando de configuración que se desea desconfigurar, por ejemplo, para borrar la configuración del rango en el eje x

    unset xrange

    De igual manera para todos los demás comandos de configuración, de modo que el comando reset que se discutió antes sería un “unset” para todas las configuraciones hechas hasta entonces.

  • Modificando el color de fondo de la gráfica (explicación sólo para Linux):
    Algunos sistemas pueden presentar el fondo de gráfica en gris, para cambiarlo es necesario ejecutar gnuplot en consola con el siguiente modificador:

    gnuplot -bg white

    Donde pondrá el fondo de gráfica blanco, se pueden especificar muchos más colores, pero este es el más demandado para las prácticas de laboratorio.

  • Creando pausas entre órdenes:
    Para que el sistema haga una pausa incluimos la orden

    pause 0.5

    Donde 0.5 es el tiempo en segundos que realiza la pausa, un 0 no realiza pausa y -1, el más útil, no avanza hasta que presionemos “enter”.

  • Creando comentarios e ignorando líneas (para archivos con órdenes, plantillas):
    Para que una línea se ignore en una serie de comandos hay que poner al principio el símbolo #, de modo que se puede utilizar tanto para realizar un comentario, como para ignorar una orden en concreto.
