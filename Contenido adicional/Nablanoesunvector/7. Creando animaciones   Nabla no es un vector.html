7. Creando animaciones

﻿
Jue, 10/07/2008 - 17:06 — grupo

Para crear animaciones vamos a hacer uso de la representación de datos y de los comandos plot '-' y pause.

La idea es la siguiente: con un binario hecho en un lenguaje de programación (por ejemplo Fortran o C) escribimos en un fichero la cabecera del documento con los settings (configuraciones) que queramos. Uno puede ser:

reset

set encoding iso_8859_1

set title 'Animaci\ón.'

set xlabel 'x'

set ylabel 'y'

set xrange [0:10]

set yrange [0:10]

set sample 5000

Seguidamente escribiremos en un bucle con tantos fotogramas como se necesite:

splot '-' w l title 'datos'

la lista de datos (tres columnas, XYZ, en el caso de splot y dos columnas, XY, en el caso de plot)

e

pause 0.05

Donde cada tanda es un fotograma de la animación. Entre plot y plot se produce una pausa dada por el comando pause.

  • Si queremos pintar dos trayectorias animadas distintas en el mismo gráfico (con distintos colores), tan solo tenemos que escribir

    splot '-' w l title 'datos1', '-' w l title 'datos2'

    lista de datos 1

    e

    lista de datos 2

    e

    pause 0.05

    Se pueden añadir tantas trayectorias como sean necesarias.

  • Para dibujar una trayectoria poco a poco (y no solo un punto) debemos usar el programa para que vaya acumulando los datos en la lista (en el fotograma 1 solo habrá un punto, en el fotograma 2 estará el primer punto y el siguiente, en el tercero estarán los dos anteriores y el nuevo, etc).
  • Un posible problema es que si no especificamos rango, en la X y la Y, se ajusta solo en función de los datos que tenga la lista.
  • Una ayuda puede ser, en el caso de representación 3D, poner el primer pause a -1, de forma que puedes orientar la gráfica como quieras en el primer fotograma y después presionar enter sobre la consola para que ocurra la animación.
